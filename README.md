## ğŸ“¦ Installation

1. Open VS Code or Cursor
2. Go to Extensions (Ctrl+Shift+X)
3. Search for "Neon Dreams Theme"
4. Click Install
5. Go to File > Preferences > Color Theme
6. Select "Neon Dreams" or "Neon Dreams Pastel"
